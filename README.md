# ScreentoPDF

**ScreentoPDF** est un outil pratique pour **prendre des captures de toutes les pages des livres que vous possédez** et qui ne sont pas encore en format PDF.

---

## Installation

1. Cloner le dépôt :  
```bash
git clone https://github.com/tonnom/autoscreenshot
cd autoscreenshot
```

2. Créer un environnement virtuel et l'activer :
```bash
python3 -m venv venv
source venv/bin/activate
```
4. Installer les dépendances :
```bash
pip install -r requirements.txt
```
5. Lancer l'interface :
```bash
python autoscreenshot_gui.py
```
